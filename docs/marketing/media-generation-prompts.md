# Media Generation Prompts

## Global art direction

Use a premium, dark, restrained, infrastructure-grade visual language. Avoid generic blue AI gradients, cartoon robots, hologram clichés, glowing orbs, cheap neon, and noisy dashboard clutter.

Preferred palette:

- charcoal and deep black backgrounds
- amber accents
- teal accents
- slate typography
- thin linework
- calm spacing
- cinematic but restrained lighting

The visuals should feel credible, high-end, and product-grade.

## Prompt 1: InfraForge systems loop

Create a premium dark product-system diagram for InfraForge. Show the relationship between InfraForge Platform, AI Inference Lab, ForgeWatch, Project Iris, Jerry, Young Coders, AI Builders Academy, and Vault Ops. The diagram should feel like a high-end infrastructure product visual, not a cloud architecture dump. Use amber and teal accents, charcoal panels, thin linework, and minimal labels. Emphasize problem-solving systems, governed AI, and reliable infrastructure. Avoid blue AI gradients and generic sci-fi visuals.

## Prompt 2: Project Iris personal AI assistant

Create a premium product mockup for Project Iris, a governed personal AI assistant for everyday digital work. Show a calm operator workspace with memory, approvals, email, calendar, browser workflows, artifacts, audit, and long-running tasks. The visual should feel like a luxury productivity product with strong privacy and user-control cues. Use dark charcoal surfaces, amber/teal accents, restrained glow, clear hierarchy, and realistic UI panels. Do not make it look like an engineering dashboard.

## Prompt 3: ForgeWatch operational intelligence

Create a premium operational-intelligence visual for ForgeWatch. Show infrastructure signals flowing from metrics, alerts, KeepHQ events, Kubernetes state, incidents, and service health into a calm triage surface. The mood should be serious, precise, and operator-grade. Use amber and teal linework on a dark background. Avoid cluttered dashboards, excessive charts, and generic blue AI styling.

## Prompt 4: Jerry mobile AI companion

Create a premium mobile product mockup for Jerry, a mobile AI companion for quick commands, status checks, approvals, alert review, and lightweight assistant interaction. Show a refined smartphone interface with dark surfaces, amber/teal accents, command chips, status cards, and approval prompts. The product should feel practical and trustworthy, not like a toy chatbot.

## Prompt 5: AI Inference Lab model operations

Create a premium model-operations visual for AI Inference Lab. Show LLM serving, benchmark signals, runtime packaging, latency, throughput, GPU capacity planning, and observability. The visual should feel like a high-end infrastructure lab. Use dark panels, amber and teal highlights, thin topology lines, and measured spacing. Avoid generic neural-network clichés and blue holograms.

## Prompt 6: Launch hero loop video

Create a 6-second premium motion loop for the InfraForge brand hub. Dark charcoal background. Thin amber and teal lines connect product-system nodes: AI Inference Lab, ForgeWatch, Project Iris, Jerry, InfraForge Platform, and education products. Motion should be slow, subtle, and cinematic. No flashy transitions, no blue AI gradients, no robots, no generic AI orb. The loop should feel like a high-trust systems company.

Slide 1: InfraForge builds governed AI systems and the infrastructure they need to operate reliably.
